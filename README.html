🎓 dAIgnostic

Adaptive Intelligent Learning System (AILS)

Sistem manajemen pembelajaran modern berbasis AI & RAG untuk pengalaman belajar yang dipersonalisasi.

Python 3.10+ FastAPI Google Gemini AI LangChain Supabase Tailwind CSS

🌟 Fitur Utama

Adaptive Learning Path

Jalur belajar yang disesuaikan secara otomatis berdasarkan kemampuan awal Anda.

AI-Generated Assessment

Soal Pretest dan Posttest yang dihasilkan secara cerdas dari materi PDF asli.

RAG Implementation

AI menjawab berdasarkan sumber materi valid, meminimalisir "halusinasi" informasi.

Smart Feedback

Laporan analisis perkembangan kompetensi dan saran karir yang mendalam menggunakan AI.

Learning Behavior Analytics

Melacak pola belajar Anda (Nibbler, Big Eater, dsb) untuk evaluasi yang lebih baik.

🚀 Panduan Setup

Persiapan Lingkungan

Pastikan Anda sudah menginstal:

  • Python 3.10+
  • Akun Supabase (Gratis)
  • API Key Google Gemini AI

Setup Backend

Buka folder backend, salin .env.example menjadi .env, lalu masukkan API Key dan URL Supabase Anda.

pip install -r requirements.txt
uvicorn main:app --reload

Setup Frontend

Buka js/api.js dan sesuaikan API_URL dengan alamat backend Anda.

// Default
const API_URL = "http://localhost:8000";
